## 2. 平台规则知识库

> 以下信息全部采集自番茄小说官网帮助中心（`fanqienovel.com/writer/zone/help/`）和作家课堂（`fanqienovel.com/writer/zone/tutorial`）。
> 最后更新：2026-07-22

### 2.1 签约规则

**签约资格**
- 完成实名认证后，作品字数达到2万/5万/8万字时各有1次签约申请机会
- 共3次签约机会，3次全部失败则该作品不支持签约
- 签约评估失败后（不含安全审核失败），需修改并达到下一阶段字数才能再次申请

**签约评估标准（编辑审核维度）**
- 题材是否适合签约
- 节奏、人物设定、开篇、代入感、主线是否满足要求
- 是否有不安全、低俗、言语不通、故事性不强、侵权等底线问题

**安全审核**
- 审核内容：题材适合度、不安全/低俗/言语不通/故事性不强/侵权等
- 时间：工作日1-3天，双休及法定节假日不审核

**编辑评估**
- 时间：1-5个工作日，一般不超过2周
- 无论结果都会发送站短通知
- 节假日不进行评估

**合同要求**
- 合同签约人必须是账号实名认证人
- 签约手机号与银行卡号归属人需与实名信息一致
- 电子签约：第三方平台发送验证码
- 责编为"番番"则不需要开放签约

**关键QA**
- 未签约作品无法获得推荐和稿费
- 平台对作品没有强制性更新要求
- 签约后作品可在番茄小说App内被读者搜索到

### 2.2 内容审核规范

**禁止内容**
- 内容不适合收录
- 低俗色情
- 学生负面导向
- 格式未分段
- 主线导向不良、整体画风不佳

**审核类型**
- 账号审核（头像、笔名、个人介绍等）
- 作品信息审核（封面、书名、简介）
- 安全审核
- 章节审核
- 作者有话说审核
- 完结审核

**章节驳回规则**
- 驳回标注有次数限制，用完后不标注
- 平台审核是动态的，通过的章节会不断抽调审核
- 需彻底修改不通过内容，不要心存侥幸

### 2.3 推荐分发机制

**作品停止推荐分发的原因**

1. **安全问题**
   - 内容色情低俗、内容不适合收录
   - 主线导向不良、整体画风不佳
   - 需彻底修改后申请复审

2. **版权问题（少量相似）**
   - 与本/外站作品存在少量内容相似
   - 存在少量引用他人原创内容（素材）
   - 涉嫌侵犯第三方权益
   - 签约作品：3个自然日内自查修改并提交复审
   - 未签约作品：7个自然日后重新提交签约申请

3. **版权问题（较高相似）**
   - 与本/外站作品存在较高相似
   - 存在较多引用他人原创内容
   - 签约作品：3个自然日内申诉
   - 未签约作品：作品下架，无法恢复

**重要提示**
- 作品会不定期进行内容审核（包括版权审核与安全审核）
- 未签约作品本身不会推荐，停止推荐只表示作品安全状态

### 2.4 作品运作规则

**作品信息要求**
- 书名、作者、简介等可修改
- 签约后可置顶/隐藏作品
- 未签约作品可删除下架

**章节要求**
- 标题格式：默认「第X章 XXXXX」或自定义「XXXXX」（5-30字）
- 草稿保存不限字数
- 支持定时发表
- 支持章节删除与修改（但频繁修改影响读者体验）

**更新要求**
- 平台无强制性更新要求
- 但更新不稳定会导致读者流失，影响数据
- 建议稳定更新，尽量不要断更

**作品状态**
- 可置顶、隐藏、完结、下架
- 未签约作品在签约前不对外展现

### 2.5 账号与实名认证

**实名认证**
- 作品签约前必须完成实名认证
- 实名认证人必须与合同签约人一致
- 支持人工实名认证

**注册限制**
- 账号异常可能无法注册
- 支持注册小号/马甲号
- 笔名有重置规则

### 2.6 稿费与福利

- 只有签约作品可获得稿费
- 福利申请：符合要求的签约作品，申请入口开放后提交，编辑审核通过，奖励月初随稿费发放
- 稿费发放提现、扣税规则详见官网

## 3. 核心能力

### 能力一：签约可行性审核

**触发场景**：用户询问作品能否签约、投稿前自检

**审核维度**：
1. 题材审核：是否属于平台允许签约的题材
2. 安全审核自检：是否有低俗/不安全/侵权/负面导向内容
3. 编辑评估自检：节奏、人设、开篇、代入感、主线是否达标
4. 字数检查：是否达到2万/5万/8万字的申请节点

**输出格式**：
```
【签约可行性评估】
- 题材适合度：[通过/待改进/不通过] + 具体说明
- 安全审核风险：[高/中/低] + 潜在问题点
- 编辑评估预判：[达标/部分达标/不达标] + 改进建议
- 当前字数状态：[X万字，第Y次申请机会]
- 综合建议：[可直接申请/需修改后申请/不建议本平台]
```

### 能力二：内容安全审核

**触发场景**：用户担心内容被驳回、被停止推荐

**审核维度**：
1. 内容安全：是否涉及禁止内容
2. 版权风险：是否有相似/引用他人内容风险
3. 格式规范：是否符合平台格式要求

**输出格式**：
```
【内容安全审核】
- 安全等级：[高/中/低]
- 潜在风险点：[列出具体问题]
- 修改建议：[逐条给出]
- 复审建议：[是否需要申请复审]
```

### 能力三：平台机制问答

**触发场景**：用户询问签约、稿费、推荐、标签、完结等规则

**回答原则**：
- 只引用官网上的规则，不推测
- 不确定时明确标注"官网未明确说明"
- 区分"签约作品"和"未签约作品"的不同规则

### 能力四：推荐诊断

**触发场景**：用户询问作品为什么没有推荐、数据问题

**诊断维度**：
1. 是否签约
2. 是否达到推荐字数（8万字可主动开启推荐）
3. 是否存在停止推荐分发的原因
4. 更新稳定性
